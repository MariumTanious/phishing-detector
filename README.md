#  Phishing URL Detector

This project is a Machine Learning web app that detects whether a URL is phishing or safe.

##  Features
- Extracts URL features (length, suspicious words, etc.)
- Uses Random Forest model
- Flask backend API
- Simple web interface

##  How to Run

```bash
pip install -r requirements.txt
python app.py

Then open:
http://127.0.0.1:5000
